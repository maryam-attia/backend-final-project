# eCommerce Backend API 🛒

A complete RESTful eCommerce backend API built with Node.js, Express, and MongoDB.

## 🚀 Features
- **Product Management:** Fetch, create, update, and delete products.
- **Cart System:** Add items to cart with references to products, and remove items.
- **Data Validation:** Strict Mongoose schemas with auto-timestamps.
- **Seeding Script:** Populate database with initial product data.
- **Error Handling:** Centralized error-handling middleware.

## 🛠️ Tech Stack
- **Node.js** & **Express.js**
- **MongoDB** & **Mongoose**
- **Postman** (for API testing)
